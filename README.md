_Market Trend Analysis (Beginner Project)_

This project focuses on basic market trend analysis using historical stock price data.  
The goal is to understand how prices move over time and identify simple trends using Python.

_Objective_
- Analyze historical stock price data
- Identify basic market trends (uptrend / downtrend)
- Visualize price movement using charts
- Build foundational understanding of market analysis

_Tools & Technologies Used_
- Python
- Jupyter Notebook
- pandas
- matplotlib
- yfinance

_Dataset_
- Historical stock price data fetched using the `yfinance` library
- Example stocks: AAPL, MSFT, or other publicly traded companies

_Methodology_
1. Download historical stock data
2. Perform basic data cleaning
3. Calculate simple indicators 
4. Visualize trends using line charts
5. Interpret trend direction based on price movement

_Results_
- Clear visualization of stock price trends
- Identification of bullish and bearish phases
- Improved understanding of how trends form in financial markets
